# Code of Conduct

CADIS follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Accept constructive feedback.
- Focus on what is best for the community.
- Show empathy toward other community members.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Sexualized language or imagery.
- Personal or political attacks.
- Publishing private information without permission.
- Other conduct that would reasonably be considered inappropriate in a professional setting.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this Code of Conduct.
