# README Media Assets

Use this folder for screenshots and small demo media referenced from the root
`README.md` or app README files.

Guidelines:

- Use kebab-case file names, for example `hud-wulan-arc.png`.
- Prefer optimized PNG, JPEG, WebP, or short MP4 files.
- Keep large videos in GitHub Releases and link them from README instead of
  committing heavy binaries.
- Do not include credentials, tokens, private paths, account names, or sensitive
  chat content in screenshots or videos.
