# CADIS Open Source

## Read First

- `docs/09_OPEN_SOURCE_STANDARD.md`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `CHANGELOG.md`
- `docs/07_MASTER_CHECKLIST.md`

## Rules

- Keep docs public-safe.
- Remove private machine paths before public release unless explicitly marked local-only.
- Keep README links current.
- Update checklist when adding work.
- Update changelog for notable changes.
- Do not add license-incompatible source or assets.

## Useful Installed Skills

- `doc`
- `gh-fix-ci`
- `gh-address-comments`
- `security-ownership-map`

## Validation

- README links point to existing files.
- No secrets in docs.
- Release docs match actual repository state.
- CI checks the files it claims to check.
